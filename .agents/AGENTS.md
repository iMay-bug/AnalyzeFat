# Regras do Projeto AnalyzeFat

- **Git Push Automático:** Sempre que concluir alterações no código ou implementar novas funcionalidades, executar o commit e fazer o `git push` (usando HTTPS para o remote origin) automaticamente ao final da tarefa, sem necessidade de confirmação extra.
- **Símbolos SVG Minimalistas em vez de Emojis:** Nunca utilizar emojis no código de UI/UX ou na interface do usuário (ex: 🔥, 🟢, 🏆, 🦾, ⚡, 📦). Em vez disso, sempre criar e utilizar símbolos e ícones via SVG inline ou componentes/strings SVG (estilo `svgTrophy`, `svgShield`, `svgLightning`), garantindo um visual muito mais limpo, profissional, organizado, discreto e coeso em todas as plataformas.
